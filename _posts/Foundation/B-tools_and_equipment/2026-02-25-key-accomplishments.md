Coding Basics I Learned

Variables: These are like containers or boxes that hold information so you can use it later. For example, you might have a variable called score to keep track of a player's points.

Functions: These are like little machines that do a specific job. You give them information (parameters), and they give you back a result.

Conditionals: These are if and else statements. They let the program make decisions, like "if the score is 10, then win the game."

Booleans: These are simple "true" or "false" values. They help the computer decide which way to go in a comparison.

Nested Conditionals: This is just putting one if statement inside another one. It’s useful for making more specific rules for your code.

Iterations: These are loops like for and while. They repeat code so you don't have to type the same thing over and over. (My team actually taught this lesson!)

Arrays: An array is a list that stores many items in one variable. You can find specific items in the list by using their index number.

Strings: Strings are used for text. You can join words together, cut them, or change how they look.

Mathematical Expressions: This is using math in code, like adding, subtracting, or using the modulo (%) to find a remainder.

JSON and JavaScript Objects: Objects store data in pairs (like "Name: Rishab"). JSON is just a way to organize that data so it’s easy to send and read.

Classes and Methods: A class is a blueprint for an object, and methods are the actions that object can do. For example, a "Car" class might have a "drive" method.

HTML DOM: The DOM is how JavaScript talks to your website. It turns the webpage into a tree of "nodes" so you can use code to change the text, colors, or images on the screen.

Data Abstraction: This is a fancy way of saying "keeping things simple." It means organizing messy data into clean objects or functions so the code is easier to use.
